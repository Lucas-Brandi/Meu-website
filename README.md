# 🌐 Meu Website (React + Vite)

Este projeto é a versão moderna do meu site pessoal, recriado com **React + Vite** a partir da versão original em HTML, CSS e JavaScript puro.  
O objetivo foi manter o mesmo visual e comportamento do site antigo, mas com uma estrutura mais organizada, modular e moderna.

---

## 🚀 Tecnologias Utilizadas

- ⚛️ **React** – Criação dos componentes e páginas dinâmicas  
- ⚡ **Vite** – Build tool rápida e moderna  
- 🧭 **React Router DOM** – Sistema de navegação entre páginas  
- 🎨 **CSS3** – Estilos originais adaptados do projeto antigo  
- 🖋️ **Google Fonts / Font Awesome** – Fontes e ícones  
- 🧩 **JavaScript (ES6+)**

---


## 💡 Funcionalidades

* Scroll suave entre seções (adaptado do app.js original)
* Layout responsivo com design escuro
* Organização modular em componentes React
* Código mais limpo, escalável e fácil de manter
